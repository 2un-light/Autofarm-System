## 🚀 주요 기능
- 수중 펌프와 솔레노이드 밸브 제어 (자동 환수)
- 양식장의 PH데이터와 양식장 수온 데이터 모니터링
- 농작물의 토양습도, 조도 모니터링 및 LED 제어
- 양어장 자동 먹이 급여

<br><br>

## 🏆 성과
- 정보통신기획원장상 수상
- 한국 정보처리 학회 논문 기재
- 교내 창업 아이디어 경진대회 최우수상
- 전국 창업/창직/창작 공모전 장려상
- 교내 캡스톤 디자인 대회 금상
